# <img src="https://raw.githubusercontent.com/EgorTatarnikov/WordByHeart/main/WordByHeart_v2.svg" width="32" height="32" alt="wt badge"> [Word by Heart](https://github.com/EgorTatarnikov/WordByHeart)
Приложение для подготовки лексики к изучению, необходимой для прочтения конкретной книги или просмотра сериала.<br>
В основе приложения лежит **NLP-программа**, которая:
- анализирует текст (**частотный анализ**, **TF-IDF**)
- выбирает **ключевые слова**
- переводит лексику **с учётом контекста**
- формирует двусторонние **карточки слов** для изучения

В результате **Word by Heart** превращает сырой текст в готовый набор слов для изучения, и формирует карточки, которые можно сразу использовать для запоминания.

Проект вырос из практической методики, где вместо абстрактной цели «выучить язык» ставится конкретная задача - прочитать книгу или посмотреть сериал.<br>
Изучение ограниченного количества слов, которые реально важны для понимания данного текста, позволяет быстро приступать к потеблению контента на английском языке.<br>
Быстрый переход к чтению любимых книг и просмотру любимых сериалов на оригинальном языке, приносит удовольствие, дает чувство прогресса и добавляет мотивации для дальнейшего изучения иностранных языков.

Ход выполнения проекта:
1. С сайта wiktionary выгружена база слов в формате json (20 Гб). Проведена десериализация и формирование базы английских слов с переводом, транскрипцией, толкованиями и примерами использования.
2. С Google Books Ngram Viewer выгружена база 1-gram по состоянию на 2012 год. Составлен частотный словарь для топ 1 миллиона английских слов.
